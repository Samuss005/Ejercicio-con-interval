# Ejercicio-con-interval
Ejercicios con interval

Práctica de JavaScript – Temporizadores y Eventos

Alumno: Samuele Isidoro Moccia

 Descripción

Este repositorio contiene la resolución de 5 ejercicios de JavaScript centrados en el uso de temporizadores (setInterval, clearInterval), manipulación del DOM y lógica de eventos.

Cada ejercicio está implementado en un archivo HTML independiente en la raíz del repositorio.

Ejercicios
Ejercicio 1: Reloj en tiempo real

Archivo: ejercicio1.html
Crear un reloj que muestre la hora actual en un párrafo con id reloj y se actualice automáticamente cada segundo sin necesidad de interacción del usuario.

 Ejercicio 2: Cuenta atrás con parada

Archivo: ejercicio2.html
Mostrar una cuenta atrás desde 10 hasta 0. Al llegar a 0, mostrar "¡Tiempo!" y detener el intervalo. Incluir botones para iniciar y parar.

Ejercicio 3: Texto parpadeante

Archivo: ejercicio3.html
Un texto que aparece y desaparece cada 500 ms. Incluir botones para iniciar y detener el efecto.

Ejercicio 4: Barra de progreso animada

Archivo: ejercicio4.html
Crear una barra de progreso que avance del 0% al 100% en 10 segundos. Al completarse, mostrar "¡Completado!".

Ejercicio 5: Parar el número

Archivo: ejercicio5.html
Mostrar un número aleatorio del 1 al 10 que cambia cada 200 ms. El usuario debe detenerlo intentando acertar el número 7.

Si acierta → "¡Ganaste!"
Si falla → "Era el Xnúmero, inténtalo de nuevo."
Notas importantes
Cada ejercicio está en un archivo HTML independiente.
Todos los archivos se encuentran en la raíz del repositorio.
Se han seguido las pistas proporcionadas para la resolución.
Es importante evitar múltiples setInterval sin detenerlos con clearInterval.

 Tecnologías utilizadas
